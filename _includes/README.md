# README

Download [Popper.js][popper] as it is required by Bootstrap 5.

See also [cdnjs][popper-cdnjs].

```bash
curl --output js/popper.min.js https://cdn.jsdelivr.net/npm/@popperjs/core@2.11.8/dist/umd/popper.min.js
```

[popper]: https://popper.js.org/
[popper-cdnjs]: https://cdnjs.com/libraries/popper.js/2.11.8
